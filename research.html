Explain about our Sample

Our participants are all AUT students spanning from various degrees. In total we have around 11 participants. This sample will provide insight into how they use Canvas for their degrees, as Canvas has only been used this year in 2022 for AUT students. The way our participants use the website should be similar, as some website depending on the class is done slightly differently but share the same main functions.

How we conduct our usability testing

We will conduct our usability by first inviting them into a study room in the library so we can conduct this test with as little outside interaction as possible that could distract our users. We will give them the consent form to read and sign with their own will. Then inside this study room we have allocated our https://canvas.aut.ac.nz/accounts/1/external_tools/106?launch_type=global_navigation participants, we will give them a general instruction on using the websites features and functions, we will keep this consistent and narrow so that we have the same test for every participant, and so we can gather data and form our analysis from it. Keeping the instructions narrow also helps with having a consistent usability test so we can gather data on a certain area of Canvas' website. During their time conducting the usability test we will be recording them, with their consent for data collection. Once they have completed our general instructions, we will ask various questions about the difficulty of performing the task, if it was confusing and which task is the hardest.

Usability Test Scenario

This usability test is to test out how easy it is to perform tasks an average University student at AUT would use. There will be several different tasks that are each unique and different. The scenario is performing simple tasks to being ready to make a written report with a group.

1. Send a message to Nathan Flanagan or Masaomi using Canvas in the Byzantine group

2. Create a to-do list task for the HCI class; and view it once it's made in the calendar

3. Go to the library in Canvas and find the referencing section for APA 7th edition referencing

4. Using Canvas, find the lab space booking website
